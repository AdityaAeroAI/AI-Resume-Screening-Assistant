# 🤖 AI Resume Screening Assistant

An AI-powered Resume Screening Assistant built using **Flowise** and **OpenRouter API**.  
This application evaluates candidate profiles and provides structured hiring feedback including fit level, strengths, weaknesses, and recommendations.

---

## 🎯 Project Objective

To build a real-world AI assistant that simulates a recruiter by:
- Analyzing candidate skills and experience
- Evaluating job suitability
- Providing structured feedback

---

## 🧠 Features

- 💬 Conversational AI interface
- 📊 Candidate evaluation (Fit Level + Match Score)
- 🧾 Structured output (Strengths, Weaknesses, Recommendation)
- 🧠 Context-aware responses (chat memory)
- 🌐 Web-based UI using Flowise Embed

---

## 🏗️ Tech Stack

- **Flowise** – Visual AI pipeline builder  
- **OpenRouter API** – LLM integration  
- **HTML/CSS** – Frontend UI  
- **JavaScript** – Chatbot embed  

---

## 🔄 Flow Design
User Input → Prompt Template → LLM → Output
↑
Memory

---

## 📝 Prompt Design

The chatbot uses a structured HR evaluation prompt to generate:

- Fit Level (Excellent / Good / Average / Poor)
- Match Score (out of 100)
- Strengths
- Weaknesses
- Final Recommendation

---

## 🚀 Deployment

The project is deployed using **Vercel** and accessible via a public link.
